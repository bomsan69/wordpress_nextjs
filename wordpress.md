# Task
NextJS를 이용해서 Wordpress의 post를 관리하는 프로그램을 만들어 주세요

## Requirements
- Wordpewss 의 api를 이용해서 워드프레스 POST를 등록 및 조회기능 구현
- 로그인, 리스트,상새,등록 4개 페이지 제작
- 로그인 페이지는 .env파일에 저장되어 있는 id,password를 가져와서 인증하는 심플한 시스템
- 리스트 페이지는 wordpress post를 최근 한달동안 등록된 모든 post를 표시, paging기능 구현, 한화면에 10개 글 표시
- 리스트 페이지의 상단에 검색 기능 구현, 등록일, 카데고리,등록자를 select 항목으로 표시
- 리스트 페이지가 처음 표시 될때에는 등록일 기준만 적용해서 표시
- 등록 패이지는 카데고리,타이틀,본문, 등록자를 등록항목으로 표시하고 ,카데고리와 등록자는 워드프로세스에서 조회해서 실렉트 박스로 표현하세요
- 디자인은 70대 노인이 사용할 서비스라고 생각하고 심플하고 직관적으로 구성하세요


## Coding Guide
- ui-ux-frontend-expert agent가 필요하면 호출해서 사용하세요
- 고객의 요구사항을 정확히 파악하고 구현 계획을 수립하세요
- 계획을 수립하면서 고객에게 확인 받아야 할 내용이 있으먄 고객에게 질문하세요
- 계획이 수립되면 수립된 계획이 고객의 요구사항을 잘 받영하는지 다신 한번 검토하세요
- 계획이 수립되면 고객애개 설명하고 확인 받으세요
- 고객이 추가 요구사항이 없고 코딩 진행을 확인하면 코딩을 진행하세요


## 추가 요구사항
- 등록된 post는 draft로 등록되어야 합니다.
- list 페이지 처음 시작시 검색의 날짜 필터에는 현재 조회 기간이 표시되어야 합니다.
- 등록창 textarea 아래에 현재의 글자수를 표시하는 기능을 추가해 주세요
- 등록자에 기본값으로 Jay Cho가 표시되게 하세요
- 카데고리에 기본값이  Sleepless 표시 되도록 해 주세요

## Media 메뉴 만들기
- post 업무를 참고해서 Media 메뉴를 만들어 주세요.
- 이미지 리스트 섬네일로 표시하고 하단에 이미지 타이틀를 표시하세요. 그 및에 삭제 버튼을 만들어서 이 버튼을 클릭하면 이미지를 삭제 할 수 있도록 해 주세요.
- 이미지 리스트 한번에 20개씩 보여주고 페이징을 하단에 추가해 주세요
- 이미지 리스트  우축 상단에 이미지 등록 버튼을 클릭하면 이미지를 등록하는 페이지로 이동
- 이미지 등록 페이지는 이미지를 드래그앤드럽 또는 파일 업로드 할 수 있는 창을 만들고 하단에 이미지 타이틀을 입력 할 수 있는 항목을 만들어 주세요
- 이미지 드래드앤드럽 창 위애 이미지 미리보기 창을 만들어 주세요.
